## Installation

```
git clone https://github.com/atess/vyral-app.git
```
```
cd vyral-app
```
```
composer install
```

#### docker
```
./vendor/bin/sail up -d
```
```
./vendor/bin/sail artisan migrate
```

## Test
```
./vendor/bin/sail artisan test
```
